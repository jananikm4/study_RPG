# 🎮 Study RPG

*A cozy little productivity game disguised as a study app.*

Study RPG transforms studying from “staring into the academic void” into a rewarding game experience. Complete study sessions, gain XP, level up, maintain streaks, unlock themes, and take care of your chaotic little mascot companion along the way.

Built with Python + Streamlit, this project combines productivity, gamification, and cozy internet aesthetics into one emotionally manipulative study experience.

---

## ✨ Features

### ⏳ Pomodoro Timer

* Customizable study and break sessions
* XP rewards for completed focus sessions
* Bonus XP for longer study streaks

### 🎯 RPG Progression System

* Gain XP by studying
* Level up as you progress
* Unlock new themes, cosmetics, and titles

### 🐾 Mascot Companion

A tiny creature that:

* celebrates your victories
* judges your procrastination
* slowly becomes emotionally attached to your productivity

Mood system includes:

* Happy
* Proud
* Sleepy
* Sad
* Chaotic

---

### 🔥 Daily Streaks

Track consistent study habits with gentle encouragement instead of guilt.

No “YOU FAILED” energy here.
We recover. We continue. We lock in.

---

### 🧩 Daily Quests

Examples:

* Complete 2 study sessions
* Study for 60 minutes
* Take a proper break
* Drink water like a functional organism

Rewards:

* Bonus XP
* Unlockables
* Temporary bragging rights

---

### 🎨 Unlockable Themes

Includes aesthetic themes like:

* Cozy Café ☕
* Cyberpunk 🌃
* Dark Academia 📚
* Retro Arcade 🕹️
* Aquarium 🌊
* Space Station 🚀

Each theme changes:

* colors
* UI styling
* atmosphere
* emotional damage levels

---

### 💾 Save System

Uses local JSON storage to save:

* XP
* Levels
* Streaks
* Study history
* Unlocks
* Theme preferences

No accounts.
No tracking.
No “please verify your email” nonsense.

# 🚀 Installation

Clone the repository:

```bash
git clone <your-repo-url>
cd study-rpg
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
streamlit run app.py
```
